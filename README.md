# KashGuard · 社区版资产管理栈

## 🪪 品牌介绍

**KashGuard = Key + Cash + Guard**  
品牌寓意以开源密码学守护“密钥（Key）”与“资产（Cash）”，通过可信的社区守护者（Guard）共建一个开放、透明、非营利的去中心化安全底座。

## 🤝 社区原则

- **开放治理**：所有设计讨论、路线决策与 RFC 全部公开，欢迎 PR/Issue。
- **非营利导向**：代码与文档以自由开源许可发布，所有资金仅用于基础设施与社区运作。
- **安全优先**：坚持审计、开源复核与公开测试流程。
- **多元协作**：兼顾 2B/2C 场景，但以开发者与用户共同定义需求为准。

## 🎯 产品线概览

### 基础设施层
- **KMS** - 密钥管理服务（优先级 P0）
- **HMS** - 硬件管理服务（优先级 P1）

### 核心技术层
- **MPC** - 多方安全计算（优先级 P0）

### 钱包产品线
- **自托管钱包** - 个人用户钱包（优先级 P1）
- **多签钱包** - 企业级钱包（优先级 P0）
- **智能钱包** - 账户抽象钱包（优先级 P1）
- **托管钱包** - 机构级托管（优先级 P0）

### 硬件产品线
- **硬件钱包** - 硬件安全设备（优先级 P2）
